# AMR Genome Analysis Pipeline

A bioinformatics pipeline for antimicrobial resistance (AMR) genome analysis.

## Objectives

- Learn Unix/Linux workflows
- Develop reproducible bioinformatics pipelines
- Practice Git and GitHub version control
- Analyze biological sequence data relevant to AMR research

## Project Structure

data/      - Raw biological data
scripts/   - Analysis scripts
results/   - Output files
docs/      - Documentation
